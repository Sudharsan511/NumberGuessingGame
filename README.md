# 🎯 Number Guessing Game

A simple Java console-based Number Guessing Game where the player tries to guess a randomly generated number within a limited number of attempts. The game offers multiple difficulty levels, score calculation, and input validation.

## 📌 Features

- Random number generation
- Three difficulty levels
  - Easy (1–50)
  - Medium (1–100)
  - Hard (1–200)
- Limited attempts
- Score calculation
- Play Again option
- Input validation
- Exception handling

## 🛠 Technologies Used

- Java
- Scanner Class
- Random Class
- Exception Handling
- Loops
- Conditional Statements

## ▶️ How to Run

1. Open the project in VS Code or any Java IDE.
2. Compile the program.
3. Run `Main.java`.
4. Select a difficulty level.
5. Guess the number within the allowed attempts.
